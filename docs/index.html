Modal Analysis of Resonant Structures

Computational modal analysis tool for resonant structures — Euler-Bernoulli FEM solver, FFT-based vibration analysis, and parametric optimization in Python.

Python 3.8+ NumPy · SciPy Matplotlib · Seaborn Euler-Bernoulli FEM FFT Signal Processing

Optimization

Minimize mass while keeping the first mode above a target frequency.

Analysis Workflow

  1. Define beam geometry and steel material properties.
  2. Compute analytical natural frequencies using Euler-Bernoulli theory.
  3. Assemble global stiffness and mass matrices via FEM; solve eigenvalue problem.
  4. Synthesize a time-domain vibration signal and extract frequencies via FFT.
  5. Validate all three methods against each other.
  6. Run mesh convergence study to confirm FEM accuracy.
  7. Perform parametric sweeps over thickness and length.
  8. Optimize beam geometry to avoid resonance with a 50 Hz pump.
